# IS5126 Final Project
## Email Classifier

### Reports
Written on LaTeX with overleaf tools. You can find the reports under `Report` folder

### Dataset
Collected base dataset from HuggingFace: `https://huggingface.co/datasets/jason23322/high-accuracy-email-classifier`, adding syntethic data using OpenAI API to expand the dataset. You can find under `Dataset` folder

### Notebooks
Every data collecting, EDA, processing, model training, creating pipeline, exporting models, external API calls simulation, and evaluation steps are documented in the notebooks under `Source Code/notebooks/`

### Running Application
We have deployed our application under https://email-manager.streamlit.app/, or can also run in the local
You can find and follow the guide in `Source Code/application/README.md`

### Slides
You can find the presentation slide under `Slides` folder
